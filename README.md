# Groupie Tracker
Groupie Tracker is a client server for the [groupie trackers](https://groupietrackers.herokuapp.com/api) API. It uses the [render-html](https://github.com/Lord-lami/render-html)'s render package to dynamically render the JSON responses from the API into HTML elements.

## Usage
Terminal:
```bash
go run .
```

Docker:
```bash
./dockerize.sh
```
