techniques - iteration, functions

solution - api data - html mapping
Automatically convert/map API endpoints to pages and the data from those endpoints into html elements

assessment - 
            concurrent io
memory-cost routineMemory*len(inputString)
time-cost   O(n/n)
clarity     Straight-Forward
novelty     still new

solution - concurrent io
1. concurrently read the neccesary banner file for the art of the characters
2. channel all the art into a result holder
3. print the output
    requirements: a table of characters and the position of their art in the banner file.
    the requirement has been met because the table is the ascii table.

implementation - concurrent io
1. concurrently read the neccesary banner file for the row art of the characters in a line
2. print all eight rows of art for each line
3. print all lines for the text.
